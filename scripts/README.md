# Scripts

This folder includes scripts for validating manifests, tracing PLM classifier outputs, and reproducing selected QC steps from stored inputs/embeddings.

Important caveat: R02A cluster generation itself is not regenerated here. R02A is documented as a retained-assignment stress split in `supplementary/R02A_retained_assignment_documentation_1_0.md`.

Recommended first command:

```bash
python scripts/validate_repository.py
```
